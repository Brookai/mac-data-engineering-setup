# MCP Server Configuration

MCP servers to activate when working in this repository.

## Required Servers

These servers are essential for repository workflows:

### github-official
- **Purpose**: GitHub API operations (repos, PRs, issues, releases)
- **Required for**: Creating releases, managing PRs, repository operations
- **Setup**: Requires `github.personal_access_token`
- **Activation**: `docker mcp server add github-official`

### notion
- **Purpose**: Create and manage Notion documentation pages
- **Required for**: Publishing documentation to team Notion workspace
- **Setup**: Requires `notion.internal_integration_token`
- **Activation**: `docker mcp server add notion`
- **Integration setup**: https://www.notion.so/my-integrations

## Optional Servers

Useful but not required:

### slack
- **Purpose**: Post announcements to #data-engineering channel
- **Use case**: Notify team of releases, major updates
- **Setup**: Requires `slack.bot_token`
- **Activation**: `docker mcp server add slack`

### database-server
- **Purpose**: PostgreSQL operations and natural language SQL
- **Use case**: Testing database configurations, SQL examples
- **Setup**: Requires `database_url` connection string
- **Activation**: `docker mcp server add database-server`

## Activation Instructions

When starting work in this repository:

1. **Check if Docker MCP Toolkit is running:**
   ```bash
   claude mcp list | grep MCP_DOCKER
   ```

2. **Activate required servers:**
   ```bash
   # GitHub (for releases, PRs)
   docker mcp server add github-official

   # Notion (for documentation)
   docker mcp server add notion
   ```

3. **Configure secrets:**
   - GitHub token: https://github.com/settings/tokens
   - Notion integration: https://www.notion.so/my-integrations
